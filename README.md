Alith’s Canvas – Art Gallery Website

Alith’s Canvas is a simple art gallery website built using HTML and CSS. It showcases different artworks with descriptions and storytelling behind each piece. The goal of this project is to present art in a clean and visually appealing way while sharing meaningful messages through each image.

Features
A clean and simple navigation bar
Gallery section displaying multiple artworks
Each artwork has a caption and description
Personal storytelling behind every piece
Social media links including Instagram, WhatsApp, and email
Lightweight and easy to run in any browser
Built With
HTML5
CSS3
Font Awesome icons
Project Structure

The project contains a single main file:

index.html

You can also add an assets folder if you want to organize images or other resources later.

How to Run the Project

To run this project locally:

Download or clone the project files
Open the folder
Double-click on index.html
It will open in your browser automatically
About the Project

This project is a personal art gallery created to express emotions and ideas through visual art. Each artwork represents themes such as identity, society, emotions, beauty, and real-life struggles. It combines both creativity and storytelling.
